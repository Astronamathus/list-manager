# List-Manager

A simple and efficient command-line List Manager built with Rust.
Easily add items, view your lists, and manage entries with a fast and lightweight interface.

# Features
- Add new items to a list
- View existing list items
- Lightweight and fast performance
- Built with Rust for reliability and efficiency


# Requirements
- Install Rust
- Install Cargo (included with Rust)

## Getting Started

### Clone the repository:
 
```rust
    git clone <your-repository-url>
    cd <your-project-folder>
``` 

### Build the project:

```rust
cargo build
```
### Run the application:

```rust
cargo run
```
### Usage

Example:
```rust
cargo run
```
Then follow the prompts in the terminal to add or view list items.
